# Creative Guidelines

## Messaging hierarchy
1. Audience pain point
2. Outcome/benefit
3. Proof (social proof, stats, logos)
4. Call to action

## Compliance
- Avoid unverifiable superlatives ("best", "guaranteed") unless backed by proof
- Do not include personal attributes or sensitive targeting language

## Fatigue signals
- CTR decreasing week-over-week
- Frequency increasing
- CPA/CAC increasing while landing page unchanged
