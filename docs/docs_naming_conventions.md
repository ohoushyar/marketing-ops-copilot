# Naming Conventions

## Goals
Consistent naming enables reliable reporting, experimentation, and cross-channel comparison.

## Campaign naming
Format:
`{yyyyq#}-{objective}-{product}-{geo}-{audience}-{variant}`

Where:
- objective: `brand`, `demandgen`, `retargeting`
- geo: `us`, `emea`, `apac`, `global`
- audience: `prospecting`, `competitor`, `customer`, `high_intent`
- variant: short descriptor like `v1`, `promo`, `test-a`

Example:
- `2026q1-demandgen-widget-us-prospecting-v1`

## Ad set / ad group naming
Format:
`{audience}-{placement}-{optimization}-{variant}`

Example:
- `lookalike-ig_feed-conversions-v2`

## Creative naming
Format:
`{concept}-{format}-{length}-{hook}-{cta}-v#`

Example:
- `painpoint-video-15s-fastsetup-demo-v3`
