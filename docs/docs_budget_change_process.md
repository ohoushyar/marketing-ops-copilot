# Budget Change Process

## Principles
- Avoid multiple major changes on the same day for the same campaign (confounds analysis).
- Prefer incremental changes unless a campaign is clearly broken.

## Thresholds
- Increase budget by max 20% per day per campaign when scaling
- Decrease budget immediately if spend is rising while conversions drop for 3 consecutive days

## Change log requirements
Every budget change must log:
- timestamp
- campaign
- old budget / new budget
- reason
- link to supporting data (report query or screenshot)
