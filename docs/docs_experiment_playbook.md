# Experiment Playbook

## When to run an experiment
Run an experiment when:
- CAC increased > 15% WoW for a stable campaign
- Conversion rate dropped > 10% WoW
- Creative fatigue is suspected (CTR down, frequency up)

## Experiment brief template
- Goal: (e.g., improve CVR by 10%)
- Primary metric: (CAC, CVR, ROAS)
- Guardrails: (Spend, volume floor)
- Hypothesis: (because...)
- Variants: control vs treatment
- Audience/geo scope
- Duration: minimum 7 days or until N conversions

## Decision rules
- Use directional decisions if volume is low; document uncertainty
- Roll back if guardrails breach for 2 consecutive days
